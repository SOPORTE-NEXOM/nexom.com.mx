# NEXOM

Sitio web estático desarrollado con HTML, CSS y JavaScript.

## Ejecutar en local

El proyecto incluye un servidor de desarrollo basado en Python y no necesita
instalar dependencias.

Desde la carpeta del proyecto ejecuta:

```powershell
python server.py
```

Después abre [http://127.0.0.1:8000](http://127.0.0.1:8000) en el navegador.
Para detener el servidor, presiona `Ctrl+C` en la terminal.

También puedes elegir otro puerto:

```powershell
python server.py --port 8080
```
